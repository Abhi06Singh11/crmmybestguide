# **App Name**: GuideBoost

## Core Features:

- Service Showcase: Clearly present e-commerce, application development, and ERP SaaS services.
- Project Portfolio: Display a gallery of successfully completed projects with descriptions.
- Client Testimonials: Display a range of testimonials, demonstrating value and social proof.
- Contact Form: Enable potential clients to submit inquiries directly.
- Quick Inquiry: Tool to understand the user needs in natural language to provide more suitable responses

## Style Guidelines:

- Dominant: White or very light gray (#FFFFFF, #F1F1F2).
- Secondary/Text: Navy blue or dark gray (#00246B, #2A3132).
- Accent: A vibrant electric or sky blue (#408EC6, #1995AD) for buttons and key links to signify innovation and attract attention.
  add green color
- Font pairing: 'Poppins' (sans-serif) for headers and 'PT Sans' (sans-serif) for body text.
- Use minimalistic line icons to represent services and key features.
- Clean, grid-based layout with ample white space to ensure readability and visual appeal.
- Subtle transitions and hover effects to enhance user interaction.
